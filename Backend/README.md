A digital platform built using the MERN stack (MongoDB, Express.js, React.js, Node.js) for managing and engaging alumni through a centralized system.
The platform enables college students get better internships and jobs opportunities through verified alumni sources, foster networking, and support communication between alumni and college students.

#FEATURES-
Secure authentication and authorization (JWT-based)
Real-time updates and search functionality
Event announcements, reunions, and job postings
